Young Postdoctoral Researcher

Universidad de Valladolid (Spain, March 2024 - July 2024)

Measuring physically motivated sizes of galaxies using Deep Learning segmentation and domain adaptation.

• Preparing an observed and a simulated dataset for galaxy segmentation

The observed galaxy stamps correspond to SDSS Stripe82 cutouts, while the mock galaxies were simulated with Imfit combining a bulge and a disc light profile, and later overlapped on a real sky image.

• Traning a segmentation model that adapts two domains

A U-Net model is trained on the source target (simulated) to optimize the segmentation loss. In a second stage, the decoder is frozen and both datasets are fed to the model. In addition to the segmentation problem (on source data only), a discriminator layer is introduced after the latent space to classify both data domains. At the same time, the encoder is trained to push both sets of embeddings together.

PhD in Astrophysics

Instituto de Astrofisica de Canarias & Universidad de La Laguna (Spain, December 2023)

Thesis Title: Uncovering the physics of galaxies with self-supervised deep learning

Supervisors:

Marc Huertas-Company, Johan Knapen

Topic: Using self-supervised Deep Learning and cosmological simulations to constrain the physics of galaxies through the properties derived from integral field spectroscopy observations.

• Review on Contrastive Learning for Astrophysics

I wrote an invited review with my supervisor on the application of contrastive learning, a self-supervised machine learning algorithm, applied to astrophysics.

• Capturing the physics of MaNGA galaxies with self-supervised deep learning

I have applied a novel self-supervised Deep Learning model to astronomical data. This was the first time contrastive learning was applied to IFU data, and only the second time in astronomy. We show that we can reduce observational effects while keeping intrinsic information from the galaxies, obtaining a more meaningful representation than other classical dimensionality reduction algorithms like PCA. This project inspired the use of this technique on other astronomical datasets.

• MaNGIA: 10,000 mock galaxies for stellar population analysis

I generated a forward modelled spatial-spectral dataset of simulated galaxies, resulting in the largest mock IFS sample ever made. For this I developed a python algorithm to match two distributions. I updated and modified a complex Python pipeline, and ran it in parallel using Linux systems.
